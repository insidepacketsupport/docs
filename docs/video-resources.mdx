# insidepacket NaaS Portal Video Resources

This section provides a curated list of video tutorials to help you understand and configure various features within the insidepacket NaaS Portal. Each video is designed to offer practical, step-by-step guidance on key networking and security operations.

## Video Guides

### 1. Create a DIA connection

This video guides you through the process of setting up a Dedicated Internet Access (DIA) connection, which provides a fast and reliable internet service with dedicated, symmetrical bandwidth. This process includes choosing a service plan, confirming availability, and configuring the connection for essential applications and cloud services.
